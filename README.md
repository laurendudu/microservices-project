## How to make it work

- clone the repository using `GitHub Desktop` or `GitHub Web`
- get into the right directory 

#### Using node
- `node index.js`
- go to `localhost:4000`

#### Using Docker
- `docker build --tag <tag-image> .`
- `docker-compose up -d`
- go to `localhost:4000`
